## 🎯 Термины методологии

| Термин | Определение | Синонимы |
| :--- | :--- | :--- |
| **Слой (Layer)** | Логический уровень структуры ТЗ, отвечающий за определённый аспект документации | Уровень, ярус |
| **Трассировка (Traceability)** | Прослеживаемость связи между требованиями и их реализацией | Матрица соответствия |
| **Артефакт** | Любой документ, файл или результат работы, порождаемый в процессе создания ТЗ | Документ, файл, результат |

## 🤖 Agents & Skills

| Термин | Определение | Пример |
| :--- | :--- | :--- |
| **Агент (Agent)** | Автономный программный компонент с определённой ролью и зоной ответственности | `Security Agent`, `Process Agent` |
| **Навык (Skill)** | Атомарная функция/скрипт, выполняемая агентом для решения задачи | `deadline-watchdog`, `encryptor` |
| **System Prompt** | Инструкция, определяющая поведение и ограничения агента | — |
| **Chain of Thought** | Последовательность логических шагов агента при принятии решений | — |
| **Human-in-the-loop** | Точка процесса, где требуется подтверждение человека | Утверждение акта ревизии |
