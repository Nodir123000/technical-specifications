## ПС-02: Управление ревизиями

| Атрибут | Значение |
|:---|:---|
| **Агент-владелец** | Process Agent + `kru-inspection-expert` |
| **Стандарт** | O'z DSt 1987:2018 §6.5.1 |

**Функции:**
- Полный жизненный цикл ревизии (Черновик → Исполнен → Архивный)
- Автоматический анализ импортированных транзакций на нарушения
- Управление составом ревизионной комиссии
- Контроль сроков через `deadline-watchdog`
- Поддержка загрузки и кэширования локальных выгрузок из Казначейства

**Входные данные:** Данные из ПС-01 (приказ), транзакции из ПС-05 (Excel)
**Выходные данные:** Акт ревизии, реестр нарушений → ПС-03, ПС-04

---

## ПС-04: Отчётность и аналитика

**Функции:**
- Генерация Актов ревизии (PDF, XLSX) по официальным шаблонам
- Сводные отчёты по нарушениям за период
- KPI-дашборд деятельности КРУ
- Годовые аналитические отчёты для руководства
- **Подготовка архивов (XLSX/PDF) для последующего экспорта во внешние системы**

**Связанные подсистемы:** ПС-02, ПС-03, ПС-05 (подготовка файлов к экспорту)

## ПС-05: Локальная синхронизация данных (ex-Интеграционная шина)

| Атрибут | Значение |
|:---|:---|
| **Агент-владелец** | Integration Agent + `local-excel-import` |
| **Стандарт** | O'z DSt 1987:2018 §6.5.4 |

**Поддерживаемые потоки данных (Excel):**

| Источник/Потребитель | Формат | Направление | Состав данных |
|:---|:---|:---|:---|
| Казначейство (Минфин) | XLSX | Импорт | Финансовые транзакции |
| Налоговая служба | XLSX | Импорт | Данные контрагентов |
| Кадастр / ГНИ | XLSX | Импорт | Имущественные справки |
| Внешние порталы | XLSX/PDF | Экспорт | Сформированные акты |

**Принципы:** Работа только с локальными путями, проверка SHA-256 файлов, 100% логирование операций с носителями.

---

## ПС-06: Безопасность и управление доступом

**Функции:**
- RBAC — ролевая модель доступа (матрица: [RBAC_Matrix.md](../../05_Agents_и_Skills/Матрица_связей/RBAC_Matrix.md))
- **Локальная аутентификация (внутренняя БД + аппаратный Token/PIN)**
- Маскирование ПИНФЛ во всех представлениях
- Журнал аудита всех действий (хранение 5 лет)
- **Проверка импортируемых файлов на целостность и безопасность**
- Fail-closed: при подозрении на вмешательство в локальную БД — блокировка всех операций
