# ТРЕБОВАНИЯ К ТЕХНИЧЕСКИМ СРЕДСТВАМ (АППАРАТНОЕ ОБЕСПЕЧЕЧЕНИЕ)

**Стандарт:** O'z DSt 1987:2018 §6.6.
**Класс системы:** КЗ-2 (Конфиденциально).

---

## 1. СЕРВЕРНОЕ ОБОРУДОВАНИЕ (ОСНОВНОЙ УЗЕЛ)

Для стабильной работы АИС КРР в закрытом контуре Минобороны рекомендуется использование серверного решения со следующими минимальными характеристиками:

|Параметр|Требование|Обоснование|
|:---|:---|:---|
|**Процессор (CPU)**|2 x Intel Xeon Silver (от 16 ядер)|Обработка аналитических запросов СитЦентра|
|**Оперативнаяпамять(RAM)**|128GBECCDDR4|КэшированиеБДPostgreSQLиработаИИ-агентов|
|**Дисковая подсистема**|4 x 1.92TB SSD NVMe (RAID 10)|Скорость обработки транзакций Казначейства|
|**Сетевой адаптер**|2 x 10Gbps SFP+|Внутренняя передача данных в ЛВС МО|
|**Блок питания**|Резервирование 1+1|Отказоустойчивость (99.9%)|

## 2. РАБОЧИЕ МЕСТА ИНСПЕКТОРОВ (АРМ)

Персональные компьютеры инспекторов должны соответствовать следующим параметрам для комфортной работы с веб-интерфейсом системы:

* **Процессор**: не ниже Core i5 (11-е поколение).

* **ОП**: 16 GB RAM.

* **Монитор**: Диагональ от 24" (Full HD) — критично для работы с таблицами ревизий.

* **Периферия**: Наличие USB-порта для аппаратного ключа (Token).

## 3. ПРОГРАММНАЯ СРЕДА (STACK)

* **ОС Сервера**: Astra Linux Special Edition (или аналогичная сертифицированная ОС).

* **СУБД**: PostgreSQL 16 (с расширением `pgaudit`).

* **Среда выполнения**: Node.js LTSC версий.

* **Браузер**: Рекомендуется Google Chrome или Яндекс.Браузер (последних версий).

## 4. ТРЕБОВАНИЯ К РАЗМЕЩЕНИЮ (ЦОД)

Сервер должен быть размещен в защищенном серверном помещении (ЦОД МО РУз), оснащенном:

1. Системой бесперебойного питания (ИБП) на 60 мин автономной работы.

1. Контролем доступа в помещение.

1. Системой газового пожаротушения.
