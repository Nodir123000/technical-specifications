# 🌐 Skill: i18n-localization

**ID:** `i18n-localization`
**Версия:** 1.0
**Владелец:** User Agent
**Стандарт:** Форматы Республики Узбекистан (UZS, даты, наименования)

---

## Описание

Обеспечивает бесшовное переключение между языками интерфейса: RU (русский), UZ-Lat (узбекский латиница), UZ-Cyr (узбекский кириллица). Применяет локальные форматы дат, валюты и чисел Республики Узбекистан.

---

## Входные параметры

```json
{
  "operation": "enum: [TRANSLATE, FORMAT_DATE, FORMAT_CURRENCY, FORMAT_NUMBER, DETECT_LOCALE]",
  "locale": "enum: [ru, uz-Lat, uz-Cyr]",
  "content": "string | object",
  "context": "enum: [UI, DOCUMENT, REPORT, NOTIFICATION]"
}
```

---

## Выходные параметры

```json
{
  "result": "string | object",
  "locale_applied": "string",
  "fallback_used": "boolean",
  "missing_keys": ["array of string (ключи без перевода)"]
}
```

---

## SLA

| Метрика | Значение |
|:---|:---|
| Время перевода строки | ≤ 5 мс |
| Время переключения всего интерфейса | ≤ 300 мс |
| Покрытиепереводами | ≥98%UI-строк |

---

## Форматы РУз

| Параметр | RU | UZ-Lat | UZ-Cyr |
|:---|:---|:---|:---|
| Формат даты | 18.04.2026 | 18.04.2026 | 18.04.2026 |
| Формат времени | 14:30 | 14:30 | 14:30 |
| Валюта | 1 500 000,00 сум | 1 500 000,00 so'm | 1 500 000,00 сўм |
| Разделитель тысяч | пробел | пробел | пробел |
| Десятичный разделитель | запятая | точка | запятая |

---

## Preconditions

* Файлы переводов загружены в кэш при старте системы

* Параметр `locale` определён (из профиля пользователя или браузера)

## Postconditions

* Интерфейс отображается на выбранном языке

* Документы генерируются на языке, выбранном при создании

* Выбор языка сохранён в профиле пользователя

---

## Error Flows

| Ошибка | Действие |
|:---|:---|
| Ключ перевода отсутствует | Показать ключ в квадратных скобках `[KEY_NAME]`, записать в `missing_keys` |
| Locale файл не загружен | Fallback на `ru`, установить `fallback_used: true` |
| Неверный формат числа/даты | Вернуть исходную строку без форматирования |

---

## Тест-кейсы

### TC-01: Переключение RU → UZ-Lat

**Вход:** Весь UI, locale=uz-Lat
**Ожидание:** Все элементы переведены, `missing_keys` пуст, переключение ≤ 300 мс

### TC-02: Формат валюты

**Вход:** `1500000.50`, locale=uz-Lat
**Ожидание:** `"1 500 000.50 so'm"`

### TC-03: Отсутствующий ключ

**Вход:** Новый UI-элемент без перевода, locale=uz-Cyr
**Ожидание:** `[NEW_KEY]` в интерфейсе, ключ в `missing_keys`, fallback_used=false
