# 🤖 Агенты и Навыки (Agents & Skills)

Этот раздел описывает интеллектуальный слой системы, который выходит за рамки классического функционального описания.

## 1. Спецификация Агентов (Agents)

Каждый агент — это автономная сущность с определенной ролью.

* **Agent Name:** Имя.

* **System Prompt:** Описание личности и ограничений.

* **Tools:** Инструменты, к которым у агента есть доступ.

## 2. Репозиторий Навыков (Skills)

Навыки — это атомарные функции или скрипты, которые агенты используют для выполнения задач.

* *Пример:* `generate-report`, `validate-security`, `check-deadline`.

## 3. Сценарии взаимодействия

Как агенты общаются между собой и с пользователем.

* *Chain of Thought:* Описание логики рассуждений.

* *Human-in-the-loop:* Точки, где требуется одобрение человека.

---
*Использование этого слоя позволяет создавать системы, которые не просто хранят данные, а активно помогают пользователю в принятии решений.*
