# 🚀 Starter Kit: Инициализация нового проекта ТЗ

Данный скрипт позволяет мгновенно развернуть структуру "9 Слоев" в новой директории.

## Инструкция:

1. Скопируйте файл `init-ts.ps1` в папку вашего нового проекта.
2. Запустите его через PowerShell.

## Скрипт (init-ts.ps1):

```powershell
$folders = @(
    "01_Нормативка/ГОСТ", "01_Нормативка/ISO", "01_Нормативка/O'zDSt", "01_Нормативка/Внутренние_регламенты",
    "02_Документы_заказчика/ТЗ_от_заказчика", "02_Документы_заказчика/Бизнес_требования", "02_Документы_заказчика/Описание_процессов", "02_Документы_заказчика/Skills_Agents",
    "03_Анализ/Разбор_требований", "03_Анализ/Gap_analysis", "03_Анализ/Риски", "03_Анализ/Ограничения",
    "04_Архитектура/Общая_архитектура", "04_Архитектура/Подсистемы", "04_Архитектура/Интеграции", "04_Архитектура/Диаграммы",
    "05_Agents_и_Skills/Skills", "05_Agents_и_Skills/Agents", "05_Agents_и_Skills/Сценарии_использования", "05_Agents_и_Skills/Матрица_связей", "05_Agents_и_Skills/Samples",
    "06_Техническое_задание/Черновики", "06_Техническое_задание/Финальная_версия", "06_Техническое_задание/Согласование", "06_Техническое_задание/Samples",
    "07_Тестирование_и_приемка/Тест-кейсы", "07_Тестирование_и_приемка/Сценарии_приемки", "07_Тестирование_и_приемка/Акты",
    "08_Документация/Пользовательская", "08_Документация/Техническая", "08_Документация/Инструкции",
    "09_Трассировка/Матрица_требований", "09_Трассировка/Связь_требований_и_функций"
)

foreach ($folder in $folders) {
    if (!(Test-Path $folder)) {
        New-Item -ItemType Directory -Path $folder -Force
        New-Item -ItemType File -Path "$folder/.gitkeep" -Force
        Write-Host "Created: $folder" -ForegroundColor Green
    }
}

Write-Host "`nРазвертывание завершено! Теперь скопируйте базовые README.md и шаблоны." -ForegroundColor Cyan
```
