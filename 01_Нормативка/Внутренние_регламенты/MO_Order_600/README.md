# Внутриведомственные приказы Министерства обороны

В данном разделе собраны регламентирующие документы МО РУз, на которых базируется логика работы АИС КРР.

## Приказ МО РУз № 600 от 2025 года

**Наименование:** "Об утверждении ведомственных документов, регулирующих деятельность Контрольно-ревизионного управления Министерства обороны Республики Узбекистан"

### Структура пакета документов:

1.  **[Основной приказ](Order_600_Main.md)** — общие положения и ввод документов в действие.
2.  **[Приложение № 1 (Положение о КРУ)](App_01_Regulation_KRU.md)** — определяет статус, задачи и структуру управления.
    *   [Приложение 1-1](App_01_Sub_1.md)
    *   [Приложение 1-2](App_01_Sub_2.md)
    *   [Приложение 1-3](App_01_Sub_3.md)
3.  **[Приложение № 2 (Инструкция по подразделениям обеспечения)](App_02_Instruction_Provision.md)** — методика контроля тыловых и технических служб.
    *   Содержит 16 дополнительных приложений (формы актов, предписаний и т.д.)
4.  **[Приложение № 3 (Инструкция по финансам)](App_03_Instruction_Finance.md)** — методика контроля финансовой деятельности.
    *   Содержит 6 дополнительных приложений.

### Значимость для АИС КРР:
*   Определяет **Ситуационный центр** как ядро аналитической деятельности.
*   Устанавливает требования к **дистанционному аудиту**.
*   Формирует регламент **электронного взаимодействия** между КРУ и объектами контроля.
