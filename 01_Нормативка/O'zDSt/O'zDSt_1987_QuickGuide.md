# O'z DSt 1987:2018 — Информационная технология. Комплекс стандартов на автоматизированные системы.

Данный стандарт является основополагающим в Республике Узбекистан для разработки технической документации на автоматизированные системы (АС).

## Ключевые разделы ТЗ по стандарту:

1.  **Общие сведения** (Наименование, шифр, заказчик/разработчик).
2.  **Назначение и цели создания системы** (Зачем и для кого).
3.  **Характеристика объектов автоматизации** (Что автоматизируем).
4.  **Требования к системе** (Функциональные, технические, по безопасности).
5.  **Состав и содержание работ** (Этапы разработки).
6.  **Порядок контроля и приемки** (Как будем сдавать).
7.  **Требования к составу и содержанию работ по подготовке объекта**.
8.  **Требования к документированию**.
9.  **Источники разработки**.

## При разработке для гос. органов:
- Требуется обязательное соблюдение структуры глав.
- Язык: русский и/или узбекский.
- Обязательно наличие раздела по защите персональных данных (O'z DSt 3532:2021).
