# Единые технологические требования к ГИС (Актуально на 2024-2026)

Комплекс требований, регулирующий создание современных госсистем в Узбекистане.

## 🛡 Кибербезопасность и данные
*   **Закон «О кибербезопасности»:** Обязательная экспертиза системы в ГУП «Центр кибербезопасности».
*   **Закон «О персональных данных»:** Все сервера с персональными данными граждан Узбекистана должны находиться физически на территории РУз.
*   **Шифрование:** Использование сертифицированных криптоалгоритмов.

## 🌐 Интерфейс и доступность
*   Соответствие стандартам ЕПИГУ (my.gov.uz).
*   Адаптивность для мобильных устройств.
*   Поддержка многоязычности (Узбекский латиница/кириллица, Русский).

## 📊 Управление данными
*   Минимизация дублирования данных между ведомствами.
*   Использование данных только из первоисточников (например, данные о физлицах только из базы МВД/ГЦП).
