# Единые технологические требования к ГИС (2024–2026)

Комплекс требований, регулирующий создание современных государственных информационных систем в Республике Узбекистан.

## 🛡 Кибербезопасность и данные

- **Закон «О кибербезопасности» (ЗРУ-764):** Обязательная экспертиза системы в ГУП «Центр кибербезопасности». См. [ЗРУ-764](../01_Законы/ZRU_764_15.04.2022_Cybersecurity.md).

- **Закон «О персональных данных» (ЗРУ-547):** Локализация серверов на территории РУз для хранения ПД граждан. См. [ЗРУ-547](../01_Законы/ZRU_547_02.07.2019_Personal_Data.md).

- **Шифрование:** Использование сертифицированных криптоалгоритмов для защиты каналов передачи данных.

## 🌐 Интерфейс и доступность

- **Соответствие стандартам ЕПИГУ:** Совместимость с порталом my.gov.uz.

- **Адаптивность:** Корректное отображение на мобильных устройствах.

- **Многоязычность:** Поддержка узбекского (латиница/кириллица) и русского языков.

## 📊 Управление данными

- **Минимизация дублирования:** Исключение повторного ввода данных, имеющихся в других ведомствах.

- **Единые источники:** Использование данных только из первоисточников (база МВД для физлиц, база ГНК для юрлиц).

- **Интероперабельность:** Поддержка стандартов обмена данными через систему «Электронное правительство».
