# Указ Президента Республики Узбекистан № УП-6300

**Дата:** 27 августа 2021 года
**Тема:** О мерах по дальнейшему совершенствованию системы государственного финансового контроля

## Основные положения для АИС КРР

1. **Реорганизация контроля:** Переход к системе мониторинга в реальном времени и предотвращение нецелевого использования бюджетных средств.

1. **Цифровизация:** Обязательная интеграция ведомственных систем с АИС "УзАСБО" (Министерство финансов).

1. **Риск-анализ:** Назначение контрольных мероприятий на основе автоматизированного анализа данных о нарушениях.

1. **Общественный контроль:** Обеспечение прозрачности через портал "Открытый бюджет".

## Влияние на разработку системы

- **Интеграция:** Поддержка API для взаимодействия с реестрами Министерства финансов.

- **Модуль Риск-анализа:** Алгоритмы оценки рисков должны учитывать критерии, установленные Счетной палатой.

- **Регистрация:** Все акты проверок должны автоматически регистрироваться в программе «Государственный аудит».

## Текст Указа

### О мерах по дальнейшему совершенствованию системы государственного финансового контроля

### Пункт 1. Новые формы контроля

Установить дополнительные формы контроля, осуществляемые Счетной палатой:

- **финансовый аудит** — оценка достоверности финансовой отчетности;

- **аудит соответствия** — проверка соблюдения нормативно-правовых актов;

- **аудит эффективности** — анализ результативности расходов и выполнения целевых показателей.

### Пункт 5. Программный комплекс «Государственный аудит»

Обеспечить запуск программы «Государственный аудит»:

- Все мероприятия финансового контроля должны регистрироваться в данной системе.

- Мероприятия, не зарегистрированные в программе, признаются незаконными.

- Результаты проверок (за исключением госсекретов) аккумулируются в системе.

### Пункт 12. База данных и камеральный контроль

Обеспечить создание базы данных системы государственного финансового контроля и автоматизированного ПО для камерального контроля.
