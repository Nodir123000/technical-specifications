# Мастер-реестр тест-кейсов АИС КРУ МО РУЗ

**Нормативная база:** O'z DSt 1987:2018 §6 — Порядок контроля и приёмки
**Статус:** Редакция для закрытого контура (Air-Gapped)

---

## Группа AUDIT: Управление ревизиями (UC-01)

| ID | Описание | Precondition | Шаги | Ожидаемый результат | Приоритет |
|:---|:---|:---|:---|:---|:---:|
| **TC-AUDIT-001** | Создание ревизии с корректными данными | Локальная аутентификация, ПИН-код введен | 1. Открыть «Ревизии» → «Создать» 2. Заполнить все поля 3. Нажать «Сохранить» | Ревизия создана, статус «Черновик», уведомление комиссии | M |
| **TC-AUDIT-002** | Попытка входа без локального ПИН-кода | Пользователь INSPECTOR | 1. Ввести логин/пароль 2. Пропустить ввод ПИН | Доступ заблокирован, запрос ПИН-кода | M |
| **TC-AUDIT-003** | Автоанализ транзакций (загруженных из Excel) | Ревизия IN_PROGRESS, Excel импортирован | 1. Открыть ревизию 2. Нажать «Запустить анализ» | `kru-inspection-expert` возвращает нарушения по локальным данным | M |
| **TC-AUDIT-004** | Предупреждение об отсутствии данных Excel | Новая ревизия, импорт не производился | 1. Создать ревизию 2. Открыть раздел анализа | Сообщение: «Требуется импорт данных Казначейства (XLSX)» | M |

## Группа IMPORT: Локальный импорт данных (UC-04)

| ID | Описание | Precondition | Шаги | Ожидаемый результат | Приоритет |
|:---|:---|:---|:---|:---|:---:|
| **TC-IMP-001** | Успешный импорт Excel (Казначейство) | Файл на носителе, хэш совпадает | 1. Выбрать файл 2. Ввести SHA-256 3. Нажать «Импорт» | 100% строк загружено, данные привязаны к ревизии | M |
| **TC-IMP-002** | Ошибка хэш-суммы | Файл изменен вручную | 1. Попытаться импортировать файл | Ошибка: «Несовпадение контрольной суммы», запись в аудит | M |
| **TC-IMP-003** | Неверный формат шаблона Excel | Файл без колонки "Сумма" | 1. Запустить импорт | Ошибка: «Шаблон не опознан», импорт прерван | M |
| **TC-IMP-004** | Импорт без проверки безопасности | Флаг security_scan: false | 1. Попытаться загрузить файл | Блокировка: «Файл не прошел проверку безопасности» | M |
| **TC-IMP-005** | Частичный импорт (ошибки в строках) | Файл с 10% некорректных дат | 1. Запустить импорт | 90% загружено, 10% в error_log (локальный файл ошибок) | S |
