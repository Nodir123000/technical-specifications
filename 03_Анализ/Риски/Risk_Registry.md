## Н.2. РЕЕСТР РИСКОВ

### Н.2.1. Технические риски

| ID | Риск | Вероят | Влияние | Рейтинг | Меры митигации |
| ---- | ------ | ------- | --------- | --------- | --------------- |
| **T-01** | Задержка серверного оборудования | 3 | 4 | **12** | Резервирование сроков, локальные закупки |
| **T-03** | Потеря данных при импорте Excel | 2 | 5 | **10** | **SHA-256 валидация**, полный бэкап перед импортом |
| **T-05** | Коррупция данных на носителе | 3 | 4 | **12** | Дублирование носителей, проверка на АРМ-фильтре |
| **T-06** | Уязвимости локальной 2FA | 2 | 5 | **10** | Аудит аппаратных токенов, логгирование попыток PIN |

### Н.2.1а. Риски ИБ (Закрытый контур)

| ID | Риск | Вероят | Влияние | Рейтинг | Меры митигации |
| --- | ---- | ------ | ------- | ------- | --------------- |
| **S-01** | Заражение через USB (Malware) | 4 | 5 | **20** | **АРМ-фильтр (Air-Gap Gateway)**, запрет автозапуска |
| **S-02** | Инсайдерская утечка (DLP) | 3 | 5 | **15** | Логирование всех операций с ФС и портами |
| **S-03** | Обход локального RBAC | 2 | 5 | **10** | Строгая проверка прав в Backend Services |
| **S-04** | Физический доступ к серверу | 2 | 5 | **10** | Опломбирование стоек, видеоконтроль, журналы |

## Н.3. ТОП-5 КРИТИЧЕСКИХ РИСКОВ (Air-Gap Edition)

### Н.3.1. Риск S-01: Заражение через USB-носитель

| Атрибут | Описание |
| ---------- | --------- |
| **Описание** | Попадание вируса/шифровальщика в закрытую сеть при импорте Excel |
| **Вероятность** | 4 (Высокая) |
| **Влияние** | 5 (Критическое) |
| **Рейтинг** | 20 (Критический) |
| **Меры митигации** | 1. Использование выделенного АРМ-фильтра для проверки. 2. Запрет записи на носители в серверах АИС. 3. Регулярное обновление баз сигнатур (ручной внос). |
| **Ответственный** | Админ безопасности |
